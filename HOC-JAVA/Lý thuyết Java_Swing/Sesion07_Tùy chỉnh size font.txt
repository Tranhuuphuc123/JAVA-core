>>>>>Khái niệm:
 - những đoạn code giao diện thg mawcjd dịnh khó kiểm soát tỉ lệ, để giải quyết ta cần tùy chỉnh kích cỡ size của nó cho phù hợp.
 vì vậy ta có các cacch căn chỉnh bằng code trong giao diện như sau:


>>>>Cách tùy chính font chữ:
 /
 // gọi hàm tạo Font
 Font font_name = new Font("name_font", Font.BOLD,20);

// tiến hành gán font vừa khởi tạo
// jtextarea, jLabel,Jlist, Jbutton...có thể thay thế (các thẻ JPANEL liên kết thui))

 Jbutton.setFont(font_name));

   

   

    ++ trong đó:
      > "name_font": kiểu font chữ cần hiển thị.
      > font.PLAIN: đây là vd đại diện font.PLAIN là font chữ thô(font chữ bình thường) ra có thể thay thế
                   + font.ITALIC
                   + font.BOLD
                   + font.... bổ sung thêm

      > size: kích cỡ chữ, 20,30,150....
