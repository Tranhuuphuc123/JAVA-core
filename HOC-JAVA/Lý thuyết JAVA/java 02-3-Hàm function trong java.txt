****************************TIỀM HIỂU VỀ CÁCH XÂY DỰNG FUNCTION HÀM TRONG LẬP TRÌNH JAVA***************************
>>>> KHÁI NIỆM VỀ FUNCTION(HÀM):
 - function(hàm) trong java Hàm (function) là một đoạn chương trình có tên, đầu vào và đầu ra. Hàm có chức năng giải quyết một số vấn đề 
 chuyên biệt cho chương trình chính. Hàm được gọi nhiều lần với các tham số khác nhau.

 - Hàm là một đơn vị độc lập của chương trình, không cho phép xây dựng một hàm bên trong một hàm khác. Một chương trình bao gồm một hoặc
  nhiều hàm. Trong đó, hàm main() là thành phần bắt buộc của chương trình.

  -------> hàm cũng có thể gọi là method nơi thực thi một chức năng riêng biệt cho chtrinh ( phương pháp chia để trị phân tán ra nhiều 
  method để dễ quản lý).


>>>> CẤU TRÚC XÂY DỰNG MỘT HÀM:

            modifier returnType nameOfMethod (Parameter List) {
               // method body
            }

     >> Trong đó:

           ++ Modifiers: Tạm dịch là phạm vi sửa đổi và truy cập( tiềm hiểu về acess modifier: public, default, protected, private...)
           ++ returnType: Là kiểu dữ liệu trả về
           ++ nameOfMethod: Là tên của hàm (method)
           ++ Parameter là các tham số đầu vào của hàm (có thể có nhiều tham số với nhiều kiểu dữ liệu khác nhau)
           ++ method body: là các mã code bên trong hàm, nội dung method thực thi chức năng mong muốn


           ==> vd: public static int tong(int x, int x){
                int tong = x+y;
                return tong;
           }
